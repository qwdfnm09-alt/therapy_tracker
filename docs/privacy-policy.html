Taalof Privacy Policy

Effective date: May 2, 2026

This Privacy Policy describes how the Taalof app handles user data in its current version. Taalof is a pre-marital compatibility assessment and counseling support app.

1. Developer and contact

App name: Taalof

Privacy contact inside the current app flow: 01212235330

2. Data the app can access or store

Taalof can store data entered by the user directly on the device. This may include:

  • Participant names
  • Participant ages
  • Jobs and education details
  • Questionnaire answers
  • Generated compatibility and marriage-readiness results
  • Language and theme preferences
  • Counseling booking details such as phone number, preferred date, message text, recommendation context, and send status

3. How the app uses this data

The app uses saved data to:

  • Continue the assessment flow inside the app
  • Rebuild the latest compatibility result
  • Generate a local PDF report
  • Keep the most recent counseling booking history visible
  • Preserve language and theme preferences

4. Data sharing and external actions

In the current version, Taalof does not upload saved profiles, questionnaire answers, or compatibility results to a developer-owned backend server.

However, if the user explicitly chooses to use an external action such as WhatsApp, SMS, phone call, PDF sharing, or printing, related content may leave the app through that user-triggered external action.

5. Data retention and deletion

Data remains on the device until the user clears it from inside the app. The app provides an in-app clear action in Settings to remove saved assessment and booking data.

6. Security

The current version stores data locally on the device using local app storage. No claim is made that this storage provides strong encryption beyond the platform defaults used by the app.

7. Children

The app is intended for adult relationship and counseling use cases.

8. Policy updates

This Privacy Policy may be updated when the app’s data handling behavior changes. The effective date at the top of this page will be updated when changes are published.
